### This is the code look like for this button

```jsx
import useDocusaurusContext from '@docusaurus/useDocusaurusContext'
import React from 'react'

const GlobalButton = () => {
  const { siteConfig } = useDocusaurusContext()

  return (
    <div>
      <a href={`/${siteConfig.themeConfig.GLOBAL_URL}`} class="button">Click Me</a>
    </div>
  )
}
export default GlobalButton

```

:::info 
Everything you want to put global variable put it inside react component 
and call it inside .MDX files
